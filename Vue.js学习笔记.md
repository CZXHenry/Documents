## 1. 前端开发基本概念
- **前端**：网站前端主要指的是网站的表现层和结构层。前端技术一般分为前端设计（网站的视觉设计）和前端开发（代码实现）。
- **前端主要语言** 前端开发主要使用html、css、JavaScript三种语言
  - **html** 超文本标记语言（hyper text markup language），是一种标签语言。主要用于实现网页中显示的文本内容、网页页面结构和简单的交互。
  - **css** 级联样式表（cascading style sheet），用来实现网页的风格设计，让网页能够呈现各种不同的视觉效果
  - **JavaScript** 一种面向对象的脚本语言，主要是用于实现网页中的各种动态功能，并且提供流畅的浏览体验。这些功能都是通过嵌入html文件中的js代码实现的。
- **DOM** 
  - **什么是DOM？** 文档对象模型(Document Object Model),是W3C制定的标准接口规范，提供了处理html/xml的接口。
  - **为什么需要DOM？** html/xml都是标签式的文本语言，文件里面的各种标签之间都存在一种层次关系。那么是否可以根据这两种文件的这个特性，利用一种方法将其抽象表示出来，以访问或修改各个标签中的内容呢？DOM就提供了这么一种方法，定义了一种树形结构来表示html/xml文件，树中的每个节点都表示文件中的文件中的一个元素或者元素的属性，并提供了相应的API用以操作这些元素。
  - **DOM跟JavaScript什么关系？** DOM是处理标签式语言的标准接口，即只定义了函数名称、函数参数和返回值（就跟OpenGL一回事），对怎么实现这些接口并没有要求。而因为DOM诞生在浏览器，因此一开始就是用JavaScript来实现的。如果用其他语言来解析文档，生成对象树，只要满足DOM标准，那么就是DOM的一种实现。
- **前端框架**
  - **什么是前端框架？** 在前端开发中，对于如何实现界面上的DOM操作或者界面交互等，每个人都可能有不同的实现。但是每个人的实现在性能上差异较大，而且很多时候前端开发都是有很多重复的设计和实现。因此为了提高各种web应用的性能和开发效率，便可以利用前端框架来进行开发。**所谓前端框架，其实就是一套解决方案，对于前端开发中各种问题（如如何修改元素属性等）给出良好的解决办法，每个前端开发者只要按照框架的规定去安排代码结构即可，避免了重复的开发，提高了开发效率。**
  - **常用的前端框架有哪些？** 三大主流框架vue.js、React、Angular，其他一些常用框架bootstrap、zepto.js。
    > **注意** 平时常听到的jQuery其实是一个库而不是前端框架
 
  - **框架（Framework）跟库（Library）有什么区别？** 库中实现的方法是开发者根据实际需求来调用的，而使用框架时则不是根据开发者的意愿来调用方法的，而是由框架去调用。实际上两者的区别就在于控制流的不同。
## 2. Vue.js
- **什么是Vue.js？** 前面已经提到，vue.js是一种前端框架，是由国人写的，三大前端框架之一。
- **Vue.js的特点？** vue是一套用于构建用户界面的渐进式框架，一方面，vue是被设计为自底向上逐层应用，其核心库只关注视图层，不仅易于上手，还便于与第三方库或既有项目整合；另一方面，当现代化的工具链以及各种支持类库结合使用时，vue也完全能够为复杂的单页应用提供驱动。
- **Vue.js的基本功能**
  - **声明式渲染** 允许采用简洁的模板语法来声明式地将数据渲染进DOM系统
  - **条件与循环** 可以利用条件或者循环简单的改变元素属性
  - **处理用户输入** 通过Vue提供的指令，可以简单的实现与用户的交互
  - **组件化应用构建** 允许开发者利用小型、独立和通常可复用的组件构建大型应用。
