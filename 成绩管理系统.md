# 成绩管理系统
## 1. 系统概述
一个基于图形界面的用于管理学生考试成绩的系统，能够实现添加、查询和统计学生考试成绩，能够导入Excel表中的成绩数据以及导出学生成绩Excel表。
## 2. 具体功能
该系统主要功能可以分为以下三大部分
### 2.1 学生成绩管理
- 添加学生在考试中的各科成绩
- 修改学生在考试中的各科成绩
- 查询学生某次考试的成绩
  - 根据学生姓名或学号进行查找
- 删除学生某次考试成绩
### 2.2 学生成绩统计
- 输入各科分数，计算班级上线人数
- 输入名次段，计算出各班总分及各科名次段人数
- 选择任意两次考试计算学生均分差
- 双上线分析
  - 输入各科成绩及总分分数，计算班级在这一个科目上的双上线人数
  - 计算在总分上线的基础上，某个科目上线人数多少，上线率为多少
- 可以统计总分或者各个科目中的全级前n名
- 统计学生的成绩变化（进退步折线图）
- 统计班级整体的成绩变化
- 统计全级的成绩变化
- 统计学生在某次考试成绩中各科目对总分的贡献率
### 2.3 学生成绩数据导入导出
- 导入Excel表中的学生成绩数据
- 能够选择不同的数据项并导出学生成绩Excel表
